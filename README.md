# GeekWave
A popular list about technology bbs

## 1、Get started

### Install @tarojs/cli

```bash
# Use npm
npm install -g @tarojs/cli

# Or use yarn
yarn global add @tarojs/cli

# Or use cnpm
cnpm install -g @tarojs/cli
```

### Install dependencies

```bash
# Use npm
npm install

# Or Use yarn
yarn

# Or use cnpm
cnpm install
```

## 2、Get working

### Wechat App

```bash
# Use npm
npm run dev:weapp

# Use yarn
yarn dev:weapp

# Open your wechat app IDE
```
